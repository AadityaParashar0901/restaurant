Sizzle Spot - PHP + MySQL Restaurant Website
--------------------------------------------

Run the following SQL in phpMyAdmin:
  - setup.sql

Default structure:
  index.php      -> Home with slideshow
  menu.php       -> Displays menu from DB
  cart.php       -> Session-based cart
  login.php      -> User login
  register.php   -> User signup
  admin.php      -> Add/delete menu items
  db.php         -> Database connection
  config.php     -> Site info

Assets folder -> img0.jpg ... img3.jpg + dish images
